# uplink-project-vue

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production (dist file)

```
npm run build
```

### To run the dist file of the frontend site

```
npm i -g serve
```

Then

```
serve -s dist
```
